# 貪吃蛇遊戲 (Snake Game) - 完整規格書

## 專案概述

這是一個單人網頁貪吃蛇遊戲，使用純 HTML5、CSS3 和原生 JavaScript 實現，無需任何外部依賴或框架。遊戲包含完整的用戶註冊與登入系統，使用瀏覽器的 localStorage 進行資料持久化。

---

## 如何運行

1. 直接雙擊 `snake.html` 檔案
2. 或在瀏覽器中打開該檔案
3. 無需安裝任何套件或設定環境

---

## 功能規格

### 1. 用戶認證系統

#### 1.1 註冊功能
- 輸入帳號、密碼、確認密碼
- 驗證規則：
  - 帳號和密碼不能為空
  - 密碼長度至少 4 個字元
  - 兩次輸入密碼必須一致
  - 帳號不能已存在
- 註冊成功後自動跳轉至登入頁面

#### 1.2 登入功能
- 輸入已註冊的帳號和密碼
- 驗證成功後進入遊戲頁面
- 顯示當前登入使用者資訊（暱稱縮寫頭像）

#### 1.3 登出功能
- 清除當前會話
- 停止遊戲迴圈
- 返回登入頁面

---

### 2. 遊戲核心功能

#### 2.1 基本玩法
| 項目 | 說明 |
|------|------|
| 移動控制 | 方向鍵（↑↓←→）或 WASD 鍵 |
| 食物 | 隨機出現的紅色圓形食物 |
| 得分 | 每吃到一顆食物得 10 分 |
| 成長 | 吃到食物後蛇身增長一格 |
| 遊戲結束 | 撞牆或撞到自身身體 |

#### 2.2 遊戲機制
- **速度系統**：每吃一顆食物，遊戲速度增加 2ms（最快 50ms）
- **碰撞檢測**：
  - 牆壁碰撞：蛇頭超出 canvas 範圍
  - 自身碰撞：蛇頭與蛇身重疊

### 視覺效果

- **背景**：深色漸層背景
- **毛玻璃**：登入/註冊卡片使用 backdrop-filter
- **霓虹發光**：蛇頭和食物具有 glow 效果
- **漸層動畫**：按鈕懸停時有上浮和陰影效果
- **淡入動畫**：頁面切換時有漸入動畫

## JavaScript 模組說明

### 用戶管理模組

```javascript
// 常數
const usersKey = 'snakeGameUsers';  // localStorage 鍵名

// 函數
function getUsers()        // 取得所有用戶陣列
function setUsers(users)   // 儲存用戶陣列
```

### 頁面切換模組

```javascript
function showLogin()       // 顯示登入頁面
function showRegister()    // 顯示註冊頁面
function showGamePage()    // 顯示遊戲頁面
```

### 遊戲核心模組

| 函數 | 說明 |
|------|------|
| `initGame()` | 初始化遊戲（蛇位置、方向、分數） |
| `generateFood()` | 隨機生成食物位置 |
| `update()` | 更新遊戲狀態（移動、碰撞檢測） |
| `draw()` | 繪製遊戲畫面 |
| `gameStep()` | 遊戲步進（調用 update 和 draw） |
| `gameOver()` | 處理遊戲結束 |
| `restartGame()` | 重新開始遊戲 |
| `keyDown(e)` | 鍵盤事件處理 |

```
          ↓
     [使用者驗證]
          ↓
     [遊戲頁面]
          ↓

    ┌───┴────┐
    │        │
[吃到食物] [撞牆/撞自己]
    │        │

[生成新食物]    ↓
    │    [重新開始按鈕]
    │        ↓
    └───→ [繼續遊戲]
```

## 使用說明

### 控制器

| 按鍵 | 功能 |
|------|------|
| ↑ / W | 向上移動 |
| ↓ / S | 向下移動 |
| ← / A | 向左移動 |
| → / D | 向右移動 |

### 遊戲策略

1. **初期**：速度較慢，適合熟悉操作
2. **中期**：隨著分數增加，速度逐漸加快
3. **後期**：快速移動，需要精確操作
4. **避免**：不要急轉彎，預先規劃路線

## 維護說明

### 修改遊戲參數

在 JavaScript 中修改以下變數：

```javascript
const gridSize = 20;     // 格子大小
const gameSpeed = 100;   // 初始速度 (ms)
// ...
```

### 修改 Canvas 大小

在 HTML 中修改：

```html
<canvas id="gameCanvas" width="400" height="400"></canvas>
```

### 修改配色

在 CSS 中修改對應的顏色值：

```css
/* 主色調 */
#00ff88  /* 霓虹綠 */
#ff4444  /* 霓虹紅 */
```
